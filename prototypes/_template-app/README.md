# App Prototype Template

Use this template after a P0 prototype is accepted and needs a maintainable project structure.

Default stage: P1-structured-static-app.

After online preview deployment, update the stage to P2-vercel-preview.

Files include package.json, index.html, src files, a smoke test, preview config, prototype.json, and this README.

Useful commands: npm install, npm run dev, npm run build, npm run smoke.

Open the app with ?debug=1 to show metadata, current mode, and last action.
