# P0 Prototype Template

Use this template for the first Chat-to-Prototype pass.

## Purpose

Create the smallest static clickable prototype that validates the core interaction.

## Files

```text
index.html
README.md
prototype.json
```

## Rules

- Prefer plain HTML, CSS, and JavaScript.
- Use static mock data.
- Do not add backend, login, database, deployment configuration, or external secrets at P0.
- Keep the first version small enough to review in ChatGPT.

## How to use

Copy this folder to:

```text
prototypes/<prototype-name>/
```

Then update:

- title and copy in `index.html`
- `name`, `path`, and assumptions in `prototype.json`
- project-specific notes in this README

## Preview

Open the GitHub HTML file through htmlpreview:

```text
https://htmlpreview.github.io/?https://github.com/JoJoJoinme/event-tracker/blob/master/prototypes/<prototype-name>/index.html
```
