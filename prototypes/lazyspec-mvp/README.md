# LazySpec MVP Static Prototype

这是一个静态原型：左侧输入自然语言需求，右侧生成轻量需求卡、默认假设、关键追问、验收标准、返工风险和可复制给 Coding Agent 的 Prompt。

## 当前版本

- 不需要 API Key
- 不需要后端
- 适合 GitHub Pages / raw preview / 本地直接打开
- 内置登录、导出、搜索、通用需求四类 demo 规则

## 原型定位

这不是完整 SDD，而是 Lazy Spec：

> 用户自然描述需求，系统自动暴露默认假设，只追问影响实现方向的问题，并生成 Agent Brief。
